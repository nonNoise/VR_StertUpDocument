■　作成工程（予想）
---------------------------------------------------------------------

1. 絵を書く

2. Blenderで立体化

3. Blenderにボーン（骨）を入れる

4. Unity(UniVRM) で読み出し、ボーンの設定を関連づける

5. VRMを出力する

6. VRM対応ソフトで動作確認する
